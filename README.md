# NEON UNO MULTIPLAYER

ネオン調のサイバーパンクなデザインが特徴の、ブラウザで遊べる4人対戦型「UNO（ウノ）」カードゲームです。
不足しているプレイヤー枠をコンピュータ（AI）に設定し、残りの枠をインターネット経由の通信対戦で遊ぶことができます。

## 🎮 主な機能と特徴
- **完全サーバーレスの通信対戦（P2P）**: WebRTC（PeerJS）を利用しており、サーバーを構築することなくクライアント同士で直接接続して対戦できます。GitHub Pages等で静的にホストするだけで動作します。
- **ハイブリッドマルチプレイ**: 「自分1人 vs COM3人」の完全ソロプレイから、「自分＋友達2人 vs COM1人」など、コンピュータの人数を自由に設定したマルチプレイが可能です。
- **モバイル＆PC自動レスポンシブ**: スマートフォンとPCの画面サイズを自動で認識し、最適なレイアウトを表示します。
  - スマホでの操作性を考慮し、手札は扇状（ファン状）に展開され、スワイプやタップでスムーズに操作できます。
  - モバイルの縦画面（ポートレート）時は、ゲーム盤面が窮屈になるのを防ぐため、自動で画面の横回転（ランドスケープ）を促す警告画面を表示します。
- **美しく動的なUI/UX**: ダークモードを基調としたネオン調のプレミアムデザイン。カードを引く/出すアニメーション、特殊カード（スキップ・リバース・ドローなど）発動時のド派手なネオンエフェクト演出が対戦を盛り上げます。
- **AI代替接続機能**: 対戦中のプレイヤーが予期せず切断された場合、コンピュータが即座にそのプレイヤーを引き継ぎ（AI代替）、他のプレイヤーのゲーム進行を妨げることなく対戦を継続できます。

## 🕹️ ゲームの遊び方（通信対戦の手順）

### A. 部屋を作成する（ホストプレイヤー）
1. プレイヤー名を入力します。
2. 「部屋を作る (ホスト)」タブを選択します。
3. **コンピュータの人数（AI）**を `0〜3人` の間で指定します。
   - 例：AIを「2人」に設定した場合、あなたと「通信対戦ゲスト1人」の計2人が人間プレイヤーとなります。
4. 「部屋を作成する」ボタンをクリックします。
5. 待機室に表示される **「部屋ID」（Peer ID）** のコピーボタンを押し、一緒に対戦したい友達に共有します。
6. 友達の接続が完了するのを待ち、全員が揃ったら **「ゲームを開始する」** ボタンをクリックします。

### B. 部屋に参加する（ゲストプレイヤー）
1. プレイヤー名を入力します。
2. 「部屋に入る (ゲスト)」タブを選択します。
3. ホストから共有された **「部屋ID」** を入力欄に貼り付けます。
4. 「部屋に入る」ボタンをクリックします。
5. ホストがゲームを開始するまで待機室でお待ちください。

## 📐 UNOコアルール

当ゲームは世界的に標準的なUNOルールを採用しています。

- **初期手札**: 各プレイヤー7枚
- **カード構成**:
  - 各色（赤、青、緑、黄）: 0〜9の数字、スキップ（Skip）、リバース（Reverse）、ドロー2（Draw 2）
  - ワイルド（Wild）: 場の色を好きな色に変更
  - ワイルド・ドロー4（Wild Draw 4）: 場の色を変更し、次のプレイヤーに山札から4枚引かせ、そのプレイヤーのターンをスキップ
- **カードの出し方**: 場のカード（捨て札の一番上）と「同じ色」または「同じ数字・記号」のカードを手札から出すことができます。ワイルドカードはいつでも出せます。
- **UNO宣言**:
  - 手札が2枚で、カードを出すことで残り「1枚」になるプレイヤーは、カードを出す前に画面上の **「UNOを宣言」** ボタンを押す必要があります。
  - UNOを宣言せずにカードを出した場合、ペナルティとして山札から自動的に **2枚** 引かされます。
- **勝利条件**: 最初に手札をすべて使い切った（0枚にした）プレイヤーが勝利となります。ゲーム終了時に各プレイヤーの残りの手札の点数が計算され、順位表（スコアボード）が表示されます。

## 🚀 GitHub Pages へのデプロイ手順

本ゲームは完全な静的Webアプリケーションであるため、GitHub Pagesにデプロイするだけで、誰でもURLからゲームにアクセスして遊ぶことができます。

1. **GitHubリポジトリを作成**:
   GitHubで新しい公開（Public）リポジトリを作成します。
2. **ファイルをアップロード**:
   作成したリポジトリに、以下のファイルを直接アップロード（または `git commit & push`）します。
   - `index.html`
   - `style.css`
   - `game.js`
   - `network.js`
3. **GitHub Pagesを有効化**:
   - リポジトリのページから **「Settings」**（設定）タブを開きます。
   - 左側メニューの **「Pages」** を選択します。
   - 「Build and deployment」セクションで、Sourceを **「Deploy from a branch」** にし、Branchで `main`（または `master`）の `/ (root)` を選択して **「Save」**（保存）をクリックします。
4. **アクセスして遊ぶ**:
   - 数分待つと、上部に `https://<あなたのユーザー名>.github.io/<リポジトリ名>/` の形式で公開URLが表示されます。
   - そのURLを対戦相手に伝えることで、PCでもスマホからでも即座に対戦を開始できます。
